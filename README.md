Lab 6 - SQL (AS & JOIN)

Objective

Is lab ka objective SQL me AS (alias) aur JOIN ka use samajhna hai.

Description

Is experiment me do tables ko combine karke data retrieve kiya gaya. AS keyword ka use karke column aur table ko short naam diya gaya, jisse query simple aur readable bani.

JOIN ka use karke related tables ke data ko ek sath display kiya gaya.

Conclusion

Is lab se hume AS (alias) aur JOIN ka practical use samajh aaya, jisse multiple tables ke data ko efficiently handle kiya ja sakta hai.
